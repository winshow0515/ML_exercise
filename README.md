# ML_exercise
練習並理解機器學習的基本原理，從最簡單的線性迴歸開始，逐步實作到 CNN、ResNet 等模型，並測試在 MNIST、CIFAR-10、CIFAR-100 等資料集上的效果

# Prerequisite
* Python 3.13.5
* PyTorch >= 2.0
* torchvision
* matplotlib
* numpy

---

## 練習內容

### 1. Linear Regression
- **資料集**：隨機生成身高與體重
- **模型**：線性迴歸 (手動實作 Loss & 梯度下降)
- **成果**：手動執行訓練迴圈，回歸線逐漸貼近資料分布  
- **心得**：親眼見證機器學習的數學直觀

### 2. MNIST
- **資料集**：手寫數字 (28x28 灰階影像)
- **模型**：CNN (Conv2d → ReLU → MaxPooling → 全連接層)
- **準確率**：98%

### 3. CIFAR-10
- **資料集**：彩色影像 (32x32, 10 類別)
- **模型**：ResNet18
- **準確率**：89.19%

### 4. CIFAR-100
- **資料集**：彩色影像 (32x32, 100 類別)
- **模型**：ResNet18
- **準確率**：60.88%

---


# 心得
我從線性迴歸開始，實際體驗了 Loss 與梯度下降的數學直觀。  
之後使用 CNN 與 ResNet 成功在 MNIST / CIFAR-10 / CIFAR-100 上達到不錯的準確率，體會到「選擇合適的模型架構與超參數」的重要性。  
未來希望能更深入理解模型設計的原理，而不只是使用現有的模組。
